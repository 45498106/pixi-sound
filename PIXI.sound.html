SoundLibrary.js, line 29

Adds a new sound by alias.

Name Type Description
alias String

The sound alias reference.
options ArrayBuffer | String | Object

Either the path or url to the source file.
or the object of options to use.

Name Type Default Description
src ArrayBuffer | String optional

If options is an object, the source of file.
autoPlay Boolean false optional

true to play after loading.
preload Boolean false optional

true to immediately start preloading.
block Boolean false optional

true to only play one instance of the sound at a time.
volume Number 1 optional

The amount of volume 1 = 100%.
useXHR Boolean true optional

true to use XMLHttpRequest to load the sound. Default is false, loaded with NodeJS's fs module.
panning Number 0 optional

The panning amount from -1 (left) to 1 (right).
complete PIXI.sound.Sound~completeCallback null optional

Global complete callback when play is finished.
loaded PIXI.sound.Sound~loadedCallback null optional

Call when finished loading.
Returns:
Type Description
PIXI.sound.Sound Instance to the Sound object.
