Class: Sound

PIXI.sound. Sound

new PIXI.sound.Sound(context, options)

Sound.js, line 15

Represents a single sound element. Can be used to play, pause, etc. sound instances.

Name Type Description
context PIXI.sound.SoundContext

The SoundContext instance.
options ArrayBuffer | String | Object

Either the path or url to the source file.
or the object of options to use. See PIXI.sound.Sound.from
Example
const foo = PIXI.sound.Sound.from('foo.mp3');
foo.play();
